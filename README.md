# String Calculator TDD Kata (Dart)

This project is an implementation of the **String Calculator Kata** using
**Test-Driven Development (TDD)** in **Dart**.

The goal of this kata is to practice writing clean, incremental code by
starting with simple cases and gradually adding functionality through tests.
